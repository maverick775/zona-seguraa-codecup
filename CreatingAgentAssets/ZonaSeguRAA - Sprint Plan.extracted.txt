Épica E1: API Routes (Backend)
ID
Historia de Usuario
Criterios de Aceptación
Tareas
SP
S1-B1
Como sistema, quiero una API para crear y consultar alertas
POST /api/alerts retorna 201 con la alerta creada; GET /api/alerts?zone_id=X retorna array de alertas
Crear /api/alerts/route.js; validar campos; insertar en Supabase; retornar alerta
3
S1-B2
Como sistema, quiero una API para registrar votos y comentarios
POST /api/votes con {alert_id, nickname, comment} inserta voto; si la suma de pesos ≥3 actualiza alert.level = 2
Crear /api/votes/route.js; lógica de escalada automática a nivel 2
4
S1-B3
Como sistema, quiero una API para actualizar el estado de una alerta
PATCH /api/alerts/[id] acepta {status, level}; solo coordinadores autenticados pueden usar esta ruta
Crear /api/alerts/[id]/route.js; validar Supabase session del coordinador
3
S1-B4
Como sistema, quiero registrar usuarios temporales
POST /api/users/temp con {nickname, zone_id, language} crea usuario con expires_at = now + 1h; retorna 409 si nickname está en uso
Crear /api/users/temp/route.js; verificar unicidad; insertar
2
S1-B5
Como coordinador, quiero autenticarme con email y password
POST /api/coordinator/login usa Supabase Auth; retorna session token; páginas de coordinador verifican el token
Usar supabase.auth.signInWithPassword; middleware de protección en rutas /coordinator/*
3

ID
Historia
Razón de diferimiento
BL-1
Red Mesh secundaria 433MHz (SX1278)
Requiere firmware adicional complejo; no agrega valor diferencial en la demo
BL-2
Auto-traducción en tiempo real con API de IA
Latencia y costo; en MVP el texto multilenguaje se maneja con texto de demostración pre-escrito
BL-3
Mapa dinámico con GPS real en los dispositivos
Hardware no tiene GPS; coordenadas hardcodeadas son suficientes para el escenario
BL-4
Recuperación de contraseña para coordinadores
No es necesaria para el concurso; coordinadores son pre-creados
BL-5
Push notifications (SMS/WhatsApp)
Requiere integración con Twilio/WhatsApp Business API; fuera del tiempo disponible
BL-6
App móvil nativa iOS/Android
El web responsive en móvil es suficiente para la demo
BL-7
Integración con C5 Jalisco
En proceso de negociación; roadmap post-CodeCup
